Git is a popular version control system mostly used as a source code management system for tracking changes in a project. Using git you can have a history of all of your changes, revert to the previous state of your project, see who changed a given file and much more. There are many great things about git, and it is easy to learn!

Git is used in several USF courses, so you are encouraged to attend the workshop.

The following topics will be covered:

  • An introduction to Version Control Systems
  • Features of git
  • Installing git
  • Creating and cloning a git repository
  • Recording changes to your repository
  • Viewing the history both in terminal and GUI
  • Undoing things
  • Working with remote repositories
  • Branching and Merging
  • and Github (hosted git)
